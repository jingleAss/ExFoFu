 Перед началом работы стоит проверить обновления проекта команда:

  git pull (скачать/обновить репрезиторий)


После добавления/обновления какого либо файла вписать в 'git CND':

1. git status (проверк, какие файлы добавленны какие нет).
2. git add <имя файла который ты изменил/добавил> (Подготавливвает файл к отправке).
2а.git add -A (добавить все файлы).
3. git commit -m "коментарий" (Опять подготовка,хуй его знает зачем без этого не работает)
4. git push (НАКОНЕЦТО БЛЯТЬ ЕБАНАЯ ОТПРАВКА НА ОБЛОКО)
